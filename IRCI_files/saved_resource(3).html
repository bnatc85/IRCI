Try IRCI Locally

  1. pip install -e ".[dev]"
  2. irci wizard to run an end-to-end quarter (symbols, quarter, optional news CSV).
  3. Or run dials individually: 
    irci trust --symbols "AAPL,MSFT,AMZN,GOOGL" --start 2025-04-01 --end 2025-06-30 --news-csv data/news.csv
irci valuation --symbols "AAPL,MSFT,AMZN,GOOGL" --as-of 2025-06-30
irci coverage --symbols "AAPL,MSFT,AMZN,GOOGL" --as-of 2025-06-30
irci liquidity --symbols "AAPL,MSFT,AMZN,GOOGL" --start 2024-01-01 --end 2025-06-30
irci composite --sentiment-csv outputs/trust.csv --quarter 2025Q2
